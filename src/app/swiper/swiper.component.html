Swiper Web Component

If you want to implement Swiper Web Component in Angular (current version 19) following the instructions, the only thing you'll get is a headache: 

 import { register } from 'swiper/element/bundle';
 register();

The only way to make it run, is to use:

 import { CUSTOM_ELEMENTS_SCHEMA } from '@angular/core';

 @Component({
   ...
   schemas: [CUSTOM_ELEMENTS_SCHEMA]
 })

Also you need to include in the index.html the Swiper bundle:

 <script src="https://cdn.jsdelivr.net/npm/swiper@11/swiper-element-bundle.min.js"></script>

But no problem, just clone this repo and check the code.
